# 👟 Adidas US Sales Dashboard (2020–2021)

> An interactive Power BI dashboard built to analyze Adidas US sales performance across products, retailers, regions, and time.

## 📌 Project Overview

This dashboard provides a comprehensive analysis of Adidas US sales data for **2020–2021**. It helps stakeholders understand sales performance, profitability, customer trends, and regional performance through interactive visualizations.

The dashboard enables users to:

- 📈 Track overall sales performance
- 💰 Analyze operating profit and profit margin
- 🌎 Compare sales across different regions
- 🛍️ Evaluate product category performance
- 🏪 Analyze retailer-wise sales
- 📅 Monitor monthly and yearly trends
- 🎯 Identify top-performing products and locations

---

## 🎯 Business Objective

The goal of this dashboard is to transform raw sales data into meaningful business insights that support better decision-making.

Key objectives include:

- Monitor total sales and profit
- Identify high-performing regions
- Compare retailer performance
- Analyze product demand
- Discover seasonal sales trends
- Improve business strategy using data-driven insights
